# CyberShield Awareness Website

CyberShield is a cybersecurity awareness website developed to educate users about phishing attacks, online scams, and safe internet practices. The website helps users understand common cyber threats and learn how to stay safe while using the internet.

## Project Objective
The main objective of this project is to create awareness among common users about cybersecurity, phishing, and online scams through simple content and interactive learning.

## Features
- Information about phishing attacks
- Awareness about online scams
- Tips for safe internet usage
- Password safety guidelines
- Interactive learning games
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Learning Outcome
Through this project, I improved my front-end development skills, website design, and interactive content development. I also learned how to design informative websites for user awareness.
